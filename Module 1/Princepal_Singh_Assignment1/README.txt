# PHP Unsplash Gallery Assignment

## Instructions

1. Download Unsplash images of your choice.
2. Resize each image to 1230px wide.
3. Save them in the `images/` folder with these names:
   - misty_mountain.jpg
   - forest_lake.jpg
   - sunrise_peak.jpg
   - green_valley.jpg

4. Create 300px wide thumbnails with names:
   - misty_mountain_thumbnail.jpg
   - forest_lake_thumbnail.jpg
   - sunrise_peak_thumbnail.jpg
   - green_valley_thumbnail.jpg

5. The gallery (`Intro to PHP.php`) will show the large images with photographer credit.
6. The lightbox (`lightbox.php`) will use thumbnails to link to the large images.

---
Add your images, no code edits needed!
